- 단순 시그널별 수익률 백테스트를 진행했을 때의 결과: US의 경우(top_n 0 run) 긍정 신호(거래량 동반 상승, 갭 상승, 신고가 등)는 시장 수익률 대비 초과수익률이 1~60일 기간 모두 양수, 부정 신호는 초과수익률이 5, 20, 60일 구간에서 전부 음수의 결과값을 도출. KR은 의미 있는 결과를 보여주지 못함.

- 실제로 통계적으로 의미가 있는지 여부를 판단하기 위해 부트스트랩을 통해 신뢰구간을 산정하여 각 시그널과 구간의 신뢰구간을 테스트함. 결과는 모든 시그널이 신뢰구간 95%를 통과하지 못함.(US, KR 모두) 즉, 단순히 긍정 신호가 나타났다고 하여 매수하고 부정 신호가 나타났다고 하여 매도하는 방법으로는 초과수익률을 얻을 수 없다는 사실을 알 수 있음. 하지만 내가 설정한 시그널 이외에 다른 유의미한 시그널이 존재할 수 있음은 유의해야함. 추후 추가 조사 이후 의미 있을것이라고 판단되면 테스트해보도록 계획함. 우선 다음은 시그널과 뉴스/공시 등 참고자료가 있었는지 여부를 결합하여 백테스트를 진행하는 게 좋을 것 같음.
